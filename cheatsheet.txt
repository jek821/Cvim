================================================================================
                           neovim cheatsheet
================================================================================

general
--------
<space>          leader key
qq               escape from insert mode
<space>w         save file
<space>q         quit
<space>?         open this cheatsheet

window navigation
-----------------
ctrl+h           move to left window
ctrl+j           move to window below
ctrl+k           move to window above
ctrl+l           move to right window

terminal mode
-------------
<space>tt        open terminal in bottom split
esc              exit terminal mode to normal mode
ctrl+q           exit terminal and close window
ctrl+h/j/k/l     navigate to other windows from terminal

file explorer (nvim-tree)
-------------------------
<space>e         toggle file explorer
ctrl+x           open file in horizontal split
ctrl+s           open file in vertical split

fuzzy finder (telescope)
------------------------
<space>ff        find files
<space>gr        live grep (search text in files)
ctrl+q           close telescope
ctrl+s           open file in vertical split

lsp (language server)
---------------------
gd               go to definition
gr               show references
K                show hover documentation
<space>rn        rename symbol
<space>ca        code actions
<space>d         show diagnostics in float
]d               go to next diagnostic
[d               go to previous diagnostic

compilation
-----------
<space>cc        compile current c file with -g flag (for debugging)

debugging (dap)
---------------
<space>b         toggle breakpoint on current line
f5               start/continue debugging
f10              step over (execute current line)
f11              step into (enter function)
f12              step out (exit function)
<space>dr        open debug repl
<space>dt        toggle debug ui

autocompletion
--------------
ctrl+space       trigger completion menu
enter            confirm selection
tab              next completion item
shift+tab        previous completion item

================================================================================
workflow: debugging c programs
================================================================================
1. open your .c file in neovim
2. set breakpoints with <space>b on lines you want to pause at
3. compile with <space>cc (this adds -g flag for debug symbols)
4. start debugger with f5, enter path to executable (e.g., ./main)
5. debug ui opens showing variables, stack, breakpoints
6. use f10 to step over, f11 to step into functions
7. inspect variables in the left panel
8. continue with f5 or quit debugger

================================================================================
tips
================================================================================
- files auto-save when you leave the buffer
- use <space>e to quickly toggle file tree
- use <space>ff to quickly find files by name
- use <space>gr to search for text across all files
- set breakpoints BEFORE starting the debugger
- always compile with -g flag when debugging (use <space>cc)
